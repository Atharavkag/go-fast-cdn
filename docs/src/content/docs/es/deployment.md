## Cómo Contribuir
1. Elige un archivo marcado como "Pendiente" (⏳)
2. Tradúcelo manteniendo la estructura de Markdown
3. Mantén los bloques de código en inglés
4. Actualiza el estado en este README
5. Envía un Pull Request

## Estructura de Archivos
Cada archivo traducido debe mantener la misma estructura y nombre que su equivalente en inglés.
