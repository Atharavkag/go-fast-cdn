# Introduction à go-fast-cdn

go-fast-cdn est un réseau de distribution de contenu (CDN) construit avec Go conçu pour servir des fichiers statiques rapidement et efficacement.

## Caractéristiques Principales
- **Haute performance**: Sert les fichiers avec une faible latence
- **Configuration facile**: Configuration minimale requise
- **Mise en cache**: Cache intelligent pour le contenu statique
- **Multiplateforme**: Fonctionne sur Windows, Linux et macOS
