# Traduction Française

Ce dossier contient la traduction française de la documentation de go-fast-cdn.

## État de la Traduction
- ✅ **intro.md** - Terminé
- ✅ **getting-started/installation.md** - Terminé
- ⏳ **configuration.md** - En attente (lien vers version anglaise)
- ⏳ **deployment.md** - En attente (lien vers version anglaise)

## Comment Contribuer
1. Choisissez un fichier marqué "En attente" (⏳)
2. Traduisez-le en conservant la structure Markdown
3. Gardez les blocs de code en anglais
4. Mettez à jour l'état dans ce README
5. Soumettez une Pull Request

## Structure des Fichiers
Chaque fichier traduit doit conserver la même structure et le même nom que son équivalent en anglais.
