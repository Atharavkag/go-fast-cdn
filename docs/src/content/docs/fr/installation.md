# Installation

Ce guide vous aidera à installer go-fast-cdn sur votre système.

## Prérequis
- Go 1.19 ou supérieur
- Git installé

## Étapes d'Installation

### 1. Cloner le Dépôt

git clone https://github.com/kevinanielsen/go-fast-cdn.git
cd go-fast-cdn
Construire l'Application
go build ./cmd/server
### 3. Configurer
Copiez le fichier de configuration exemple:
cp config.example.yaml config.yaml
### 4. Exécuter
./server
Prochaines Étapes
Configurez votre premier site web

Consultez les options de configuration
